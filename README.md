# Connect the Dots - Valentine's Edition ❤️

A simple and romantic web application to celebrate Valentine's Day.

## Purpose

This project is designed to tell someone special that you love them and ask: **"Will you be my Valentine?"**

## Features

- **Connect the Dots Game**: Reveal a hidden message by connecting the dots.
- **Background Music**: Plays "All 4 Nothing" by Lauv (starts after user interaction).
- **Interactive Proposal**: A final screen with a "Yes" button that triggers a celebration.
